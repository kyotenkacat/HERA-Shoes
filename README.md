[HERA Shoes](https://kyotenkacat.github.io/HERA-Shoes/)
===
模擬販售女鞋的電商網站

- 資策會個人作品 (2018.09 CD105期)
- 僅使用原生 HTML, CSS, JavaScript
- 具有微動畫效果及使用者互動功能
- 支援 RWD (mobile, pad, desktop)
- 搜尋、分頁、篩選、登入/註冊、購物車按鈕皆為模擬示意，無實際功能
